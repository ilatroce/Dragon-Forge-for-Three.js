# Dragon Forge for Three.js

> A small, interactive forge for creating procedural dragons in the browser.

Dragon Forge is a visual playground for developers exploring creature generation with **Three.js**. It combines seeded anatomy, modular traits, expressive silhouettes, and curated color palettes to produce **10,000 reproducible dragons**—each ready to inspire your own procedural systems.

## Inside the forge

- Seeded, repeatable dragon generation
- Modular heads, wings, horns, armor, tails, and body shapes
- Multiple palettes with animated presentation
- Orbit, zoom, auto-rotate, and wireframe inspection
- One self-contained browser build—no installation required

## Run it

Open [`index.html`](./index.html) directly in a modern browser, or serve the folder locally:

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080` and forge a dragon.
